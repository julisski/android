Network Parsing, hands-on

Network Parsing is a small Android app with one job: make it obvious how an app talks to a network, parses the JSON, and shows the result safely. It fetches notes with Retrofit + kotlinx.serialization and renders explicit Loading / Success / Error states. This page lets you run that fetch yourself — choose the outcome, watch the request travel the layers and come back as a UI state — then explore the DTO→domain mapping, the offline switch, the architecture, and the source.

Note.kt — DTO + domain + mapper
@Serializable                                  // makes this class JSON-decodable
data class NoteDto(
    val userId: Int,                           // matches JSON "userId" (parsed, then dropped)
    val id: Int, val title: String, val body: String,
)

// PLAIN data class: no @Serializable, no JSON knowledge, and NO userId.
data class Note(val id: Int, val title: String, val body: String)

// The SINGLE place network-shape becomes app-shape.
fun NoteDto.toNote(): Note = Note(id = id, title = title, body = body)

NoteApi.kt — the Retrofit interface + assembly
interface NoteApi {
    @GET("posts")                              // relative path -> HTTP GET
    suspend fun fetchNotes(): List<NoteDto>    // JSON array -> List<NoteDto>, parsed for you
}

private val json = Json { ignoreUnknownKeys = true }   // new server fields won't break parsing

fun provideNoteApi(): NoteApi =
    Retrofit.Builder()
        .baseUrl("https://jsonplaceholder.typicode.com/")          // MUST end in "/"
        .addConverterFactory(json.asConverterFactory("application/json".toMediaType()))
        .build()
        .create(NoteApi::class.java)           // Retrofit generates the implementation

NoteRepository.kt — Real, Fake, switch
interface NoteRepository { suspend fun getNotes(): List<Note> }   // may THROW on failure

class RealNoteRepository(private val api: NoteApi = provideNoteApi()) : NoteRepository {
    override suspend fun getNotes() = api.fetchNotes().map { it.toNote() }   // network + parse + map
}

class FakeNoteRepository : NoteRepository {
    private val json = Json { ignoreUnknownKeys = true }
    override suspend fun getNotes(): List<Note> {
        delay(600)                                              // imitate latency (see Loading)
        return json.decodeFromString<List<NoteDto>>(FAKE_NOTES_JSON).map { it.toNote() }
    }
}

// THE SWITCH — one argument toggles offline (default) vs live.
fun provideNoteRepository(useFake: Boolean = true): NoteRepository =
    if (useFake) FakeNoteRepository() else RealNoteRepository()

NotesViewModel.kt — sealed state + loadNotes()
sealed interface NotesUiState {
    data object Loading : NotesUiState
    data class Success(val notes: List<Note>) : NotesUiState
    data class Error(val message: String) : NotesUiState
}

class NotesViewModel(private val repository: NoteRepository = provideNoteRepository()) : ViewModel() {
    private val _uiState = MutableStateFlow<NotesUiState>(NotesUiState.Loading)
    val uiState: StateFlow<NotesUiState> = _uiState.asStateFlow()
    init { loadNotes() }

    fun loadNotes() {
        viewModelScope.launch {
            _uiState.value = NotesUiState.Loading
            try { _uiState.value = NotesUiState.Success(repository.getNotes()) }
            catch (e: Exception) { _uiState.value = NotesUiState.Error(e.message ?: "Something went wrong while loading notes.") }
        }
    }
}

MainActivity.kt — collect + exhaustive when
@Composable
fun NotesScreen(modifier: Modifier = Modifier, viewModel: NotesViewModel = viewModel()) {
    val uiState by viewModel.uiState.collectAsStateWithLifecycle()
    NotesContent(uiState, onRetry = viewModel::loadNotes, modifier)
}

@Composable
fun NotesContent(uiState: NotesUiState, onRetry: () -> Unit, modifier: Modifier = Modifier) {
    when (uiState) {                                           // exhaustive: sealed type
        is NotesUiState.Loading -> LoadingState(modifier)
        is NotesUiState.Success -> NotesList(uiState.notes, modifier)
        is NotesUiState.Error   -> ErrorState(uiState.message, onRetry, modifier)
    }
}

Key idea · the parsing boundary

DTO → domain mapping

JSON arrives in the network's shape (NoteDto); the app uses its own clean shape (Note). The mapper toNote() is the single boundary — it copies the fields the UI needs and drops the rest. Toggle the JSON to see two robustness behaviors.
